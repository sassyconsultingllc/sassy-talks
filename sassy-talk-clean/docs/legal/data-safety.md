### Detailed Explanations

#### Audio (Transient)
- **What:** Voice audio during push-to-talk
- **Why:** Core app functionality - walkie-talkie communication
- **Storage:** NOT stored - transmitted in real-time only
- **Encryption:** AES-256-GCM end-to-end encryption
- **Sharing:** NOT shared with third parties

#### Device ID (Local Only)
- **What:** Randomly generated identifier
- **Why:** Peer discovery on local network
- **Storage:** Device memory only, regenerated on reinstall
- **Sharing:** Visible only to devices on same local network

### Security Practices

✅ Data is encrypted in transit (AES-256-GCM)
✅ No data transmitted to external servers
✅ No third-party analytics or advertising SDKs
✅ No persistent storage of communications
✅ User can delete all data by uninstalling app

### Data Deletion

Users can request data deletion by uninstalling the application. Since no data is stored on external servers, uninstallation removes all app data.
