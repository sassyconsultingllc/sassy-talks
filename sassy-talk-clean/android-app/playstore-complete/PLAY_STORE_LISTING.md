## Full Description (4000 chars max)

Sassy-Talk is a secure push-to-talk walkie-talkie app that works over your local WiFi network. All voice communication is encrypted with AES-256-GCM -- the same encryption standard used by the military. No internet connection, servers, or accounts required.

FEATURES:

- AES-256-GCM Encryption: Military-grade encryption for all voice transmissions
- Push-to-Talk: Hold to talk, release to listen -- just like a real walkie-talkie
- 99 Channels: Separate encrypted channels for different teams or groups
- WiFi Direct: Connect directly to nearby Android devices without a WiFi router
- WiFi Multicast: Works across Android, iOS, and Windows on the same WiFi network
- Bluetooth Fallback: Legacy Bluetooth RFCOMM support when WiFi is unavailable
- QR Code Key Exchange: Secure key pairing via QR code scan -- no passwords to remember
- Zero Data Collection: No accounts, no analytics, no telemetry, no tracking
- No Internet Required: Everything stays on your local network
- Built in Rust: Memory-safe native code for reliability and performance

HOW IT WORKS:

1. Open Sassy-Talk on two or more devices on the same WiFi network
2. Scan the QR code to exchange encryption keys
3. Select a channel (1-99)
4. Hold the PTT button to talk, release to listen

PERFECT FOR:

- Construction sites and job sites
- Events and conventions
- Security teams
- Outdoor activities (camping, hiking)
- Warehouses and large facilities
- Any situation where you need quick, private voice communication without relying on cell service or internet

PRIVACY:

Sassy-Talk collects zero data. No accounts, no registration, no analytics. Your voice never leaves your local network. Encryption keys are exchanged directly between devices via QR code and never transmitted to any server. See our full privacy policy at https://sassyconsultingllc.com/privacy-policy.html

TECHNICAL SPECS:

- Encryption: AES-256-GCM (Galois/Counter Mode)
- Key Exchange: ECDH with X25519 curve
- Transport: UDP multicast over local WiFi (239.255.42.42:5555)
- Language: Pure Rust with Android JNI
- Minimum Android: 8.0 (API 24)
- Install Size: ~28 MB
