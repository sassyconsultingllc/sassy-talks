# Sassy-Talk iOS Native

iOS implementation placeholder for PTT walkie-talkie.

## Status

🚧 **Under Development**

The iOS version will use:
- Swift/SwiftUI for UI
- Rust core via UniFFI or Swift bindings
- CoreBluetooth for device discovery
- AVAudioEngine for audio

## Planned Features

- Push-to-talk voice communication
- Bluetooth LE discovery
- WiFi UDP multicast audio
- End-to-end encryption
- Background audio support

## Requirements (Planned)

- Xcode 15+
- iOS 14.0+
- Rust 1.70+ (for core library)
